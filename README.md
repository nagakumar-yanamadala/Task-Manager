# Task Manager

A simple and responsive **Task Manager App** built using **HTML, CSS, and JavaScript**.  
Users can easily add, complete, and delete tasks — all in the browser!

---

##  Live Demo

 [Live Site](https://taskmanager365.netlify.app)

---

##  Features

-  Add new tasks
-  Mark tasks as complete
-  Delete tasks from the list
-  Clean and responsive UI

---

##  Built With

- HTML5
- CSS3
- JavaScript
